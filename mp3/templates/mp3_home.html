Music Project

Screen Shots

Motivation

I run with an old iPhone and have to store my music directly on the phone. For this reason, I developed an interface to easily download new audio from YouTube (which can include lectures and audiobooks). Different artists and songs can be searched with the iTunes API along with short previews of their music.

Pages

Music Queue

Lists the entries that will be downloaded overnight. Use hyperlinks to edit/remove entries.

Search Music

Search for songs either by song or artist name (see attribute input). A preview of a song can be played using the playback icon. The hyperlink is used to automatically fill a form to add the song to the queue. Use the "Search on YouTube" link to open YouTube results for that artist/song.

Add Manually

Opens the same form for adding to the queue. This can be used for audio not found on the iTunes API.

Implemented

  • Make downloading YouTube audio faster
    • Have control over mp3 property tags
  • Schedule scripts to download music queue directly to iTunes library

Future work

  • Implement more search tools using the iTunes API to help discover new/related music
  • Add scripts that automatically create/add to playlists based on music properties like bpm, genre.
