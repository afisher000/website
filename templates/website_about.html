Website Project

Motivation

I've been interested in having a personal website for several years, but wanted to be able to develop and host the website myself. After the COVID chip shortage, I was finally able to buy a Raspberry Pi, purchased a domain name, and started to develop this site. It has been a fun project and made me even more interested in the power of data. I look forward to expanding this tool and transitioning to a more powerful home server in the future.

Goals

  • Explore PHP and JavaScript
  • Build website infrastructure using the Laravel code framework
  • Design webpages using Bootstrap CSS classes
    • Make design functional and somewhat aesthetic
  • Gain experience working on a Linux machine
  • Achieve proficiency in MySQL Databases
    • Using Workbench GUI
    • Using command line
  • Restrict access to database and private pages to admin users
  • Develop good backup habits
  • Develop good logging habits

Future Work

  • Migrate other existing projects to website/database.
  • Move server to a more powerful machine.
